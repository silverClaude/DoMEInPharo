# DoME In Pharo

**Paste the following code in playground and do it.**

To test the DoMESpecification class:

```smalltalk
ds := DoMESpecification new.

ds open.
```



To test the DoMELayout class:

```
lo := DoMELayout new.

lo open. 
```



To test the Menu class:

```smalltalk
m := Menu new.

m topMenuTest.
```


To test the TopToolbar

```smalltalk
x := TopToolbar new.

x open.
```
